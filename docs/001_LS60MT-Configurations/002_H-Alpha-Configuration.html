The LS60MT configured for observing and imaging the Sun in the H-Alpha spectrum. Be sure to read the description below the image.

Two filters must be installed for safe use: the B1200 blocking filter and the pressure-tuned etalon filter. Both are easy to recognize:

  • The B1200 blocking filter is located inside the orange and white diagonal marked with the USA flag.
  • The etalon filter is the red module with the "LUNT SOLAR SYSTEM" plate and the large pressure-tuning knob.
